About me

Hi, my name is Stephen and I'm a Web Developer. I have expertise in front-end development, but also full-stack development using the PERN stack (PostgreSQL, Express, React, and Node).

I'm comfortable with semantic HTML, CSS Grid and Flexbox, SASS, JavaScript ES6+, React (with Hooks), Git, npm and much more.

I also have an array of web experience outside of development including SEO, web accessibility, analytics and user experience.
